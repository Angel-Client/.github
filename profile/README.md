## Sobre

O **Angel Client** é um client de Minecraft para a versão **1.8.9**, desenvolvido com foco em **otimização** e performance. Nosso objetivo é entregar o melhor FPS e a experiência mais fluida possível, sem abrir mão de estabilidade.

## Destaques

- **Alta performance** — Otimizações profundas para o melhor FPS
- **Leve e estável** — Baixo consumo de recursos
- **Focado em 1.8.9** — Desenvolvido especificamente para essa versão
- **Atualizações constantes** — Melhorias e correções frequentes
